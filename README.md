# Retail Sales & Profit Analysis Dashboard

## 📊 Project Overview

This project analyzes retail sales data to uncover insights on profitability, customer segments, and discount impact using SQL and Power BI.

## 🛠 Tools Used

* SQL (Data Analysis)
* Power BI (Dashboard & Visualization)

## 📈 Key Insights

* High discounts reduce profitability and can lead to losses
* Technology category contributes the highest profit
* Certain products generate losses despite strong sales
